# 交換券価値最適化ツール

ゲーム内の工場生産ラインを最適化し、交換券価値 (pt/m) を最大化する単一HTMLツール。
ブラウザで `交換券最適化ツール.html` を直接開いて使用する。

## 構成

- DATA セクション: 資源・機械・レシピ・交換アイテムの定義。ここを編集して調整する
- LOGIC セクション: `computeFlow()` が仕様の正（評価器）
- OPTIMIZER セクション: 区分線形モデル抽出 + LP厳密解（連続値）/ 枝刈り付き格子全探索（1/3刻み・整数）

## プロジェクトルール

- Mermaidを利用して図を作っても良い
- 仕様変更は `computeFlow()` を正として行い、OPTIMIZER の自己検証（verifyModel）が通ることを確認する
